# Имитация работы диспетчера НПС

## Описание проекта
Программа генерирует отчёт о работе насосной станции (НПС) за смену на основе данных из CSV-файла.

### Функциональность:
- Чтение данных о работе насосов из CSV-файла
- Расчёт времени работы каждого насоса
- Вычисление общего перекачанного объёма
- Генерация отчёта в формате JSON

## Установка и запуск

### Требования
- Python 3.7+
- Стандартные библиотеки (csv, json, datetime, collections)

### Запуск
```bash
python main.py
```
### Формат входных данных
CSV файл (pumps_data.csv)
```csv
pump_id,start_time,stop_time,flow_rate_m3h
PUMP-001,2024-01-15 08:00:00,2024-01-15 10:30:00,150.5
PUMP-002,2024-01-15 08:30:00,2024-01-15 11:00:00,200.0
```
### Колонки:
- pump_id - идентификатор насоса (строка)
- start_time - время включения (формат: YYYY-MM-DD HH:MM:SS)
- stop_time - время выключения (формат: YYYY-MM-DD HH:MM:SS)
- flow_rate_m3h - расход жидкости в м³/час (число)
### Формат выходных данных
JSON отчёт (pump_report.json)
```json
{
  "report_info": {
    "generated_at": "2026-03-25 14:48:47",
    "total_pumps": 3,
    "total_sessions": 8
  },
  "summary": {
    "total_operating_time_hours": 22.0,
    "total_pumped_volume_m3": 3848.125,
    "average_flow_rate_m3h": 174.9148
  }
}
```
### Алгоритм работы
Чтение CSV: Парсинг файла с данными о работе насосов
Расчёт длительности: Вычисление времени работы каждой сессии
Расчёт объёма: Умножение расхода на время работы
Агрегация: Группировка данных по насосам
Статистика: Подсчёт общих показателей
Экспорт: Сохранение результатов в JSON
### Особенности реализации
Поддержка различных форматов времени
Автоматическая обработка перехода через полночь
Точное округление числовых значений (4 знака после запятой)
Подробная статистика по каждому насосу
Обработка ошибок и валидация данных
